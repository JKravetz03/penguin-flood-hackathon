Sistema Inteligente de Alerta Temprana de Inundaciones

¿Como funciona S.I.A.T.I.?

Buscamos alertar a la poblacion sobre la crecida del rio. El sistema recoje informacion de las estaciones hidrometricas y envia un mensaje a las personas registradas via WhatsApp.

Las alarmas se activan cuando el rio alcanza ciertos niveles, y son los siguientes:
Alerta Amarilla - 4,50 metros: el rio esta subiendo, ha salido de los parametros normales.
Alerta Naranja - 5 metros: el agua podria afectar a cierto numero de viviendas.
Alerta Roja - 6 metros: la inundacion compromete a un gran sector de la poblacion de los bañados.

Chat Bot

El chatbot es el asistente que se comunica con los usuarios a través de mensajes de WhatsApp. En muchas ocasiones, se convierte en un compañero virtual que se integra en sitios web y aplicaciones conversando y ayudando a los usuarios.

El sistema está programado para que interactúe con el cliente, pero sin que haya una persona física contestando. Tienen la ventaja de que están disponibles siempre para resolver las dudas de los usuarios.

A.P.I.

Las API son mecanismos que permiten a dos componentes de software comunicarse entre sí mediante un conjunto de definiciones y protocolos.

API significa “interfaz de programación de aplicaciones”. Esta se puede considerar como un contrato de servicio entre dos aplicaciones que define cómo se comunican entre sí mediante solicitudes y respuestas.

Mapas

Tenemos un mapa a disposicion que muestra las zonas aledañas al rio, en la cuidad de Asuncion, que corren riesgo de inundacion cuando el mismo pasa los 5 metros de altura. Esto ayuda a los usuarios a identificar sus barrios y la posibilidad de tener incovenientes ante una crecida.
